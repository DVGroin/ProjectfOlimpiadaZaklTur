## 📋 Описание

Данный проект решает задачи, поставленные в олимпиаде, т.е. обработка с помощью неиросетевых алгоритмов звуковой информации для помощи учёному Михаилу в определении инопланетных рас.

## 🚀 Быстрый старт

### 1. Требования
- Python **3.9 – 3.11** (версии 3.12+ могут быть несовместимы с TensorFlow)
- pip >= 23.0
- Git

### 2. Установка зависимостей
```bash
# Создание виртуального окружения (рекомендуется)
python -m venv venv
# Активация:
# Windows:
venv\Scripts\activate
# Linux/Mac:
source venv/bin/activate

# Установка пакетов
pip install -r requirements.txt --only-binary :all:
